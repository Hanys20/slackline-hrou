TEXTURY A POZADÍ (volitelné)
============================
Formát: SVG nebo WebP
Použití: jemné textury pozadí sekcí, organické tvary

Soubory (volitelné):
  blob-green.svg          – zelený organický tvar (citátová sekce)
  blob-beige.svg          – béžový organický tvar (dekorace)
  texture-paper.webp      – jemná papírová textura (volitelné přes ::before)
  pattern-leaves.svg      – pattern listů jako subtle pozadí sekce

Poznámka:
  Web funguje i bez těchto souborů – jsou čistě dekorativní.
  Organické tvary jsou nyní generovány přes CSS border-radius.
